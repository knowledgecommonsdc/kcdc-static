Crash Course in Ruby on Rails

date Saturday, 18 May 2013 time 10 a.m. to 3 p.m.

👤 Andy Fiedler, Sean Perkins 🏢 Affinity Lab

 Past

Ruby on Rails is an open-sourced web application framework using the popular programming language Ruby. Some of today’s fastest-moving companies — Twitter, Hulu, and Living Social among them — use Ruby on Rails to quickly create powerful websites.

In this four-hour class (with an hour-long break for lunch), you’ll learn how web applications work; how to use developer tools like integrated development environments and version control; and how to push a finished application to a cloud hosting solution.

A good understanding of HTML is necessary to get the most out of this class. If you don’t know the difference between an HTML <div> and an HTML <span>, this isn’t the class for you. A bit of programming knowledge in is helpful, but not required.

Students will need to bring a laptop capable of running a virtual machine (at least 2GB RAM and 10GB free hard disk space).

Instructors

Andy Fiedler is a professional software developer and technology consultant — in other words a huge nerd. He has worked on Wall Street, for the federal government, and now for himself as a freelancer in D.C. He’s really excited to help more people understand how to build awesome things with technology.

Sean Perkins has been a technologist his entire life, taking things apart long before he knew how to put them back together. Prior to running a professional web development shop, he served in the US Army and consulted for the auto industry. He enjoys making technology accessible through informal training and great design.
